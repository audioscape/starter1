Cookies


The site/js/embed.js file includes a copy of the js-cookie library.
js-cookie is a simple, lightweight JavaScript API for handling browser cookies.

About js-cookie
https://github.com/js-cookie/js-cookie

How to get and set cookies:
Cookies.get('sitemode');
Cookies.set('sitemode', 'hero');
Cookies.remove('sitemode');

The embed.js script sets:
a_access_token - Azure access token
golog - Page to redirect to after login
searchParams - Multiple parameters from current search filter
sitemode - The type of header. "hero" for image with navigation. "widget" for image only.
sitelook - The theme: Dark for Android (default). Light for iOS (coi).

Cookie values within the current domain
Most of these come from the browser and sources other than embed.js
